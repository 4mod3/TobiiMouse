## Build
### For Linux:
```bash
# Clone the repository.
git clone https://github.com/lhy0403/TobiiMouse
cd ./TobiiMouse/src/
mkdir build && cd build

# QMake and make
qmake ../
make

# Done, if everything goes well, you should have successfully built it.
```
